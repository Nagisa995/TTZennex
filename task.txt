Создать многофункциональный компонент выбора по аналогии <select> из HTML.

Пример реализации можно подсмотреть, например, здесь: https://mui.com/material-ui/react-select/#default

Необходимо реализовать следующие возможности компонента:
    • Передача массива объектов с указанием для каждого элемента списка его названия и иконки (опционально). По желанию можно добавить другие параметры.
    • Параметр для возможности ввода текста с фильтрацией предлагаемых элементов, а не только выбор из списка. Один из примеров реализации: https://react-select-search.com/?path=/story/multiple--search
Можно вынести инпут с поиском в отдельный блок с выпадающем списке.
    • Настройка стиля компонента (фон, цвет выбранного элемента, цвет текста, размер шрифта, тип границы и т.д.). Не обязательно делать много параметров, можно ограничиться некоторым готовым набором, как в Bootstrap (пример large, simple, colorful...).
    • Возможность множественного выбора.

Использовать чистые JS, Typescript и React.js, без сторонних библиотек. Стили необходимо также написать свои. Желательно использование одного из CSS-препроцессоров (SCSS/SASS/LESS).
Хорошим тоном будет создание страницы с документацией на GitHub.
Уделить внимание дизайну и удобству использования (UI&UX).
